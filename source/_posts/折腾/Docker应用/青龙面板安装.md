# docker 安装

注意 latest 不一定是最新的

```bash
docker run -d \
--name qinglong \
--hostname qinglong \
--net br0 \
--ip '10.1.1.100' \
-p 5700:5700 \
-v /mnt/user/appdata/qinglong/data:/ql/data \
whyour/qinglong:latest
```
